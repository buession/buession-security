# Buession Security 文档

该站点基于 ydoc 构建而成，网站内容在 src 文件夹内，格式为 Markdown。欢迎 issue 或 pull request。

## 开发

```bash
$ npm install ydoc  # or yarn ydoc
$ cd security.buession.com
$ ydoc serve
```

## 部署

本仓库使用的是 Github Actions 分支进行自动部署。
