# Buession Security

基于 Pac4j、Shiro、Spring Security 和极验的安全类库，封装了多种密码加密方式。

- JDK 1.8+

## Introduction

- git clone https://github.com/buession/buession-security
- cd buession-security/buession-security-parent && mvn install
